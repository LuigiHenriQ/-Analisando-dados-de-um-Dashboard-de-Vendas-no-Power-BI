# 📊 Relatório de Vendas e Seguimentos - Power BI

Este repositório contém um dashboard desenvolvido no Power BI com foco em análise de vendas e acompanhamento de indicadores de desempenho. O objetivo é transformar dados em insights úteis para a tomada de decisão estratégica.

## 🎯 Objetivo

Monitorar os principais indicadores de vendas e acompanhamento de clientes, com visualizações que ajudam a identificar padrões de desempenho, oportunidades de melhoria e tendências comerciais.

## 🔍 Funcionalidades

- Visão geral de vendas totais por período
- Acompanhamento por região, produto ou equipe
- Análise de desempenho individual e por time
- Identificação de gargalos ou quedas de performance
- Comparativo entre metas e resultados alcançados

## 🧠 Tecnologias Utilizadas

- Power BI Desktop
- Modelagem de dados e DAX
- Transformação de dados com Power Query
